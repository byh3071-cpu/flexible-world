# ADR-0001: 철학 헌장을 코드 머지 게이트로 격상

- **Status**: Accepted
- **Date**: 2026-05-25
- **Deciders**: 🤔 철학자, 🏗️ 메인 아키텍트
- **Tags**: governance, philosophy

## Context

`.cursorrules` 파일에 「개발자의 4가지 절대 원칙」(이신론적 신 / 스스로 증명 / 강제 없는 책임 / 피투성)이 정의되어 있으나, 본 원칙을 위반한 코드(V14 사법 시스템)가 실제로 머지되었다. 즉 **원칙이 문서로만 존재하고 게이트로 동작하지 않았다.**

이 프로젝트의 정체성은 곧 위 4가지 원칙이다. 원칙이 깨지면 이 프로젝트는 단순한 또 하나의 2D 멀티플레이어 게임이 된다. 따라서 원칙을 코드 머지 게이트(PR 체크리스트)로 격상한다.

## Decision

1. 4대 원칙을 `CLAUDE.md` §1 「철학 헌장」으로 격상한다. (`.cursorrules` 내용을 흡수)
2. PR 머지 게이트(`CLAUDE.md` §4-3)에 **"철학 헌장 위반 없음" 체크박스**를 필수로 추가한다.
3. 모든 에이전트는 답변 시작에 자기 페르소나를 명시하며, 🤔 철학자 모드는 트리거 시 무조건 활성화된다.
4. 현재 위반 코드(V14 사법 시스템 등)는 「위반 사례 추적」 표로 가시화하고, 별도 ADR로 처리 계획을 세운다.

## Consequences

### Positive
- 비전(노션 SoT)과 코드의 일관성 강제.
- 신규 에이전트(Claude·Cursor·기타)가 머지 전에 자동으로 원칙을 점검.
- "왜 이 게임은 다른 게임과 다른가?" 라는 정체성 질문에 답할 수 있게 됨.

### Negative
- 일부 "편한" 기능(자동 모더레이션, 운영자 명령어, RPG 직업 시스템 등)이 영구 금지됨.
- 게이트 통과 비용이 약간 증가.

### Mitigations
- 위반 사항을 "철회"하기 전에 ADR로 충분히 논의한다(ADR-0002 V14 처리 등).
- 회색지대(예: 평판 점수가 자동 격리에 영향 주는 것은 OK인가?)는 케이스별로 ADR을 남긴다.
